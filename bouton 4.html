Dérivées et primitives courantes

Dérivées

La dérivée d'une fonction f(x) représente la variation instantanée de cette fonction par rapport à la variable x. Elle est notée f'(x) ou df/dx.

  • (x^n)' = n*x^(n-1) (n étant un réel)
  • (k)' = 0 (k étant une constante)
  • (e^x)' = e^x
  • (ln(x))' = 1/x
  • (sin(x))' = cos(x)
  • (cos(x))' = -sin(x)
  • (tan(x))' = 1 / cos(x)^2
  • (arcsin(x))' = 1 / sqrt(1 - x^2)
  • (arccos(x))' = -1 / sqrt(1 - x^2)
  • (arctan(x))' = 1 / (1 + x^2)
  • (sinh(x))' = cosh(x)
  • (cosh(x))' = sinh(x)
  • (tanh(x))' = 1 / cosh(x)^2
  • (arcsinh(x))' = 1 / sqrt(1 + x^2)
  • (arccosh(x))' = 1 / sqrt(x^2 - 1)
  • (arctanh(x))' = 1 / (1 - x^2)
