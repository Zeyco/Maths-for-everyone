Primitives

La primitive d'une fonction \( f(x) \) est une fonction \( F(x) \) telle que \( F'(x) = f(x) \). Il existe une infinité de primitives pour une fonction donnée, qui ne diffèrent que d'une constante.

  • ∫\( x^n \)dx = \( \frac{{x^{n+1}}}{{n+1}} \) + C (n ≠ -1)
  • ∫\( k \)dx = \( kx \) + C (k étant une constante)
  • ∫\( e^x \)dx = \( e^x \) + C
  • ∫\( ln(x) \)dx = \( x \cdot ln(x) - x \) + C
  • ∫\( sin(x) \)dx = \( -cos(x) \) + C
  • ∫\( cos(x) \)dx = \( sin(x) \) + C
  • ∫\( tan(x) \)dx = \( -ln(cos(x)) \) + C
  • ∫\( arcsin(x) \)dx = \( x \cdot arcsin(x) + \sqrt{1 - x^2} \) + C
  • ∫\( arccos(x) \)dx = \( x \cdot arccos(x) - \sqrt{1 - x^2} \) + C
  • ∫\( arctan(x) \)dx = \( x \cdot arctan(x) - \frac{1}{2} \cdot ln(1 + x^2) \) + C
  • ∫\( sinh(x) \)dx = \( cosh(x) \) + C
  • ∫\( cosh(x) \)dx = \( sinh(x) \) + C
  • ∫\( tanh(x) \)dx = \( ln(cosh(x)) \) + C
  • ∫\( arcsinh(x) \)dx = \( x \cdot arcsinh(x) - \sqrt{1 + x^2} \) + C
  • ∫\( arccosh(x) \)dx = \( x \cdot arccosh(x) - \sqrt{x^2 - 1} \) + C
  • ∫\( arctanh(x) \)dx = \( x \cdot arctanh(x) + \frac{1}{2} \cdot ln(1 - x^2) \) + C
